# 车辆检查管理系统用户使用手册

## 1. 文档背景

本系统用于危险品运输企业的车辆检查管理，帮助企业对车辆、检查项目、检查记录、到期提醒和数据导出进行统一管理。

在日常管理中，车辆数量一旦增加，仅依赖纸质台账或普通表格，通常会出现以下问题：

- 车辆资料分散，查询不方便
- 检查项目较多，容易遗漏
- 到期时间不好追踪，容易错过检查
- 历史记录不完整，后续追溯困难
- 导出汇总麻烦，统计效率低

本系统的目标，就是把这些工作集中到一个离线桌面程序中完成，帮助管理人员更稳定、更清晰地完成车辆检查管理。

## 2. 适用对象

本手册适合以下人员使用：

- 车辆安全管理人员
- 车辆档案管理人员
- 日常检查登记人员
- 负责车辆年审、保险、营运证、罐检等周期项目管理的工作人员

## 3. 系统特点

- 本地离线运行，不依赖互联网
- 所有数据保存在本机
- 支持车辆管理、项目管理、检查记录管理
- 自动计算下次到期时间
- 支持到期提醒和逾期提醒
- 支持按条件查询和 Excel 导出
- 支持试用和激活机制

## 4. 首次使用前说明

### 4.1 软件运行方式

本系统是桌面程序，需要安装或直接运行桌面应用后使用。

### 4.2 数据保存说明

系统中的车辆信息、项目配置、检查记录、提醒记录等，都保存在当前电脑本地。

请注意：

- 更换电脑后，本机数据不会自动带到新电脑
- 卸载或清理本地数据前，请先做好导出或备份

### 4.3 试用与激活说明

系统支持试用和激活。

- 未激活时，可以在试用期内正常使用
- 试用期剩余时间会在系统界面中显示
- 试用期结束后，需要输入激活码才能继续使用
- 激活成功后，系统会自动重启并进入正常状态

如果界面右上角显示：

- `未激活`：说明当前仍处于试用或待激活状态
- `已激活`：说明当前电脑已经完成激活

## 5. 首次启动操作流程

### 5.1 打开软件

启动软件后，进入系统主界面。

首次使用时，建议先观察以下两个位置：

- 右上角状态标记：查看当前是否已激活
- 顶部提示区域：查看试用剩余时间或试用状态

### 5.2 试用期内的表现

如果当前电脑还没有激活，但仍在试用期内，系统会：

- 正常允许进入程序
- 显示未激活状态
- 显示试用剩余时间

### 5.3 试用期结束后的表现

如果试用期已经结束，系统会弹出激活窗口。

这时需要：

1. 复制本机指纹
2. 发送给软件提供方
3. 获取激活码
4. 输入激活码并完成激活

如果不激活，则无法继续使用系统。

## 6. 激活操作说明

### 6.1 获取本机指纹

当系统弹出“软件激活”窗口后：

1. 查看指纹输入框中的本机指纹
2. 点击“复制”按钮
3. 将复制出来的指纹发送给软件提供方

### 6.2 输入激活码

收到激活码后：

1. 在激活码输入框中粘贴激活码
2. 点击“激活”
3. 等待系统验证

### 6.3 激活成功后的结果

激活成功后，系统会：

- 显示激活成功提示
- 自动重启程序
- 重启后进入已激活状态

### 6.4 激活失败的常见原因

- 激活码输入错误
- 激活码与当前电脑不匹配
- 激活码已经过期或失效

如果激活失败，请重新核对：

- 是否复制了完整激活码
- 是否使用了当前电脑对应的激活码

## 7. 主界面说明

系统主界面分为两部分：

- 左侧导航栏
- 右侧功能区

### 7.1 左侧导航栏

主要包含以下菜单：

- 首页总览
- 车辆管理
- 检查项目
- 检查记录
- 查询导出

### 7.2 右侧顶部区域

顶部区域通常显示：

- 当前页面标题
- 当前日期
- 激活状态
- 刷新按钮

### 7.3 顶部试用提示

当系统处于未激活状态且仍在试用期内时，顶部会显示：

- 当前状态
- 试用总时长
- 剩余可用时间
- 试用到期时间

## 8. 首页总览使用说明

首页总览主要用于快速掌握当前车辆检查情况。

### 8.1 首页主要内容

- 统计卡片
- 已逾期车辆项目
- 7 天内到期项目

### 8.2 已逾期车辆项目

这里会显示已经超过检查时间但尚未完成处理的项目。

管理人员可以重点关注：

- 哪一台车已经逾期
- 哪个项目逾期
- 逾期多少天

### 8.3 7 天内到期项目

这里会显示未来 7 天内即将到期的项目，便于提前安排检查工作。

### 8.4 刷新数据

如果数据有变化，但首页尚未更新，可以点击右上角“刷新数据”按钮，重新加载当前页面数据。

## 9. 车辆管理操作说明

车辆管理页面用于维护车辆基础资料。

### 9.1 新增车辆

操作步骤：

1. 进入“车辆管理”
2. 点击“新增车辆”
3. 填写车辆信息
4. 保存

建议填写的信息包括：

- 车辆编号
- 车牌号
- 车型
- 负责人
- 负责人电话
- 购买日期
- 备注

### 9.2 编辑车辆

操作步骤：

1. 在车辆列表中找到目标车辆
2. 点击对应的编辑按钮
3. 修改信息
4. 保存

### 9.3 复制车辆

复制车辆适合新增同类型车辆时使用。

操作步骤：

1. 找到已有车辆
2. 点击“复制”
3. 在复制后的表单中修改车辆编号、车牌号等信息
4. 保存

复制后通常会保留原车辆的项目配置，减少重复录入工作。

### 9.4 删除车辆

如果车辆不再需要管理，可以删除。

删除前请注意：

- 删除车辆后，其关联的项目数据和历史记录可能一并删除
- 删除操作通常不可恢复

建议在删除前先确认该车辆资料是否仍需保留。

### 9.5 查看车辆项目

在车辆管理中，可以查看某一辆车当前绑定的检查项目。

你可以看到：

- 项目名称
- 周期
- 最近检查时间
- 下次到期时间
- 当前状态

### 9.6 给车辆关联项目

如果某辆车需要增加新的检查项目：

1. 进入车辆详情区域
2. 点击“关联项目”
3. 选择要关联的项目
4. 确认保存

## 10. 检查项目管理操作说明

检查项目管理用于统一维护所有检查项目。

### 10.1 可管理内容

每个检查项目通常包含：

- 项目名称
- 周期值
- 周期单位
- 是否启用

### 10.2 新增检查项目

操作步骤：

1. 进入“检查项目”
2. 点击“新增项目”
3. 输入项目名称
4. 设置周期值
5. 选择周期单位
6. 设置是否启用
7. 保存

### 10.3 编辑检查项目

如果某个项目的周期或启用状态有变化：

1. 找到目标项目
2. 点击编辑
3. 修改内容
4. 保存

### 10.4 删除检查项目

删除项目前请先确认：

- 是否仍有车辆正在使用该项目
- 删除后是否会影响已有管理流程

建议谨慎删除，优先考虑停用而不是直接删除。

## 11. 检查记录管理操作说明

检查记录页面用于录入和维护每一次实际检查结果。

### 11.1 新增检查记录

操作步骤：

1. 进入“检查记录”
2. 选择车辆
3. 选择检查项目
4. 选择检查日期
5. 选择检查结果
6. 填写备注
7. 点击“保存记录”

### 11.2 检查结果说明

通常可以选择：

- 合格
- 不合格

### 11.3 备注填写建议

建议在以下情况下填写备注：

- 存在特殊情况
- 本次检查发现异常
- 需要补充说明处理意见

### 11.4 修改检查记录

如果录入后发现信息错误，可以编辑对应历史记录。

修改后，系统会根据最新记录重新计算下次到期时间。

### 11.5 删除检查记录

如果确认某条记录录入有误且不需要保留，可删除。

删除前建议再次确认，因为删除后可能会影响项目的最近检查时间和下次到期时间。

## 12. 查询导出操作说明

查询导出页面用于筛选数据和输出报表。

### 12.1 常见查询方式

你可以按以下条件查询：

- 车辆
- 项目
- 时间范围
- 即将到期
- 本月需检查
- 已逾期

### 12.2 查询步骤

1. 进入“查询导出”
2. 选择筛选条件
3. 点击查询
4. 查看结果表格

### 12.3 Excel 导出

如需导出报表：

1. 在查询结果页完成筛选
2. 点击导出按钮
3. 选择保存位置
4. 保存 Excel 文件

适合导出的内容包括：

- 检查记录表
- 到期车辆清单
- 查询结果汇总表

## 13. Excel 批量导入说明

系统支持从 Excel 批量导入车辆。

### 13.1 使用场景

当车辆数量较多，不方便逐条手工录入时，可以使用 Excel 导入。

### 13.2 操作步骤

1. 进入“车辆管理”
2. 点击“Excel 批量导入”
3. 选择准备好的 Excel 文件
4. 检查预览结果
5. 确认无误后执行导入

### 13.3 导入前建议

- 先核对 Excel 内容是否完整
- 避免车辆编号、车牌号重复
- 导入前保留原始文件备份

## 14. 提醒规则说明

系统会根据检查周期自动计算提醒状态。

### 14.1 提醒类型

- 7 天内到期：提示即将到期
- 已逾期：提示已超过应检查时间

### 14.2 首页提醒

首页会集中展示：

- 已逾期车辆项目
- 7 天内到期项目

### 14.3 提醒消除方式

当对应车辆项目录入新的检查记录后，系统会重新计算下次到期时间，原有到期或逾期提醒会随之更新。

## 15. 日常推荐使用流程

建议按以下顺序使用系统：

1. 打开首页，查看即将到期和已逾期项目
2. 进入车辆管理，核对车辆和项目配置
3. 完成实际检查后，及时录入检查记录
4. 在查询导出中按条件筛选并导出需要的报表
5. 定期备份导出的关键数据

## 16. 常见问题

### 16.1 为什么右上角显示“未激活”？

说明当前电脑尚未完成激活，或者仍处于试用状态。

### 16.2 为什么已经过了试用期，系统不让继续使用？

因为试用期结束后需要激活码才能继续使用。请复制本机指纹并联系软件提供方获取激活码。

### 16.3 为什么激活后还提示激活？

可能原因包括：

- 激活码与当前电脑不匹配
- 激活码输入错误
- 本地激活状态异常

建议重新打开软件后再次确认，如仍有问题，请联系软件提供方。

### 16.4 为什么查询不到记录？

请检查：

- 是否选择了错误的时间范围
- 是否筛选了错误的车辆或项目
- 该记录是否尚未保存成功

### 16.5 删除的数据能恢复吗？

通常不能直接恢复。建议在执行删除前确认数据是否仍需保留。

### 16.6 软件换电脑后还能直接用吗？

通常不能直接沿用原电脑的激活状态。新电脑需要重新进入试用或重新激活。

## 17. 使用建议

- 每次检查完成后及时录入记录，避免遗漏
- 车辆基础信息变更后及时更新
- 重点关注首页中的逾期和即将到期项目
- 删除数据前先确认是否需要保留历史记录
- 定期导出关键数据作为备份

## 18. 联系与支持

如果在使用过程中遇到以下情况，建议联系软件提供方：

- 激活失败
- 激活码无效
- 数据显示异常
- 软件无法启动
- 查询结果与预期不符
- 需要重新激活或重新部署

联系时建议一并提供：

- 当前问题现象
- 操作步骤
- 截图
- 本机指纹（如与激活有关）
